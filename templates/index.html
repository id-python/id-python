Tentang Python

Python merupakan bahasa pemrograman yang memungkinkan pemrogram bekerja lebih cepat dan efektif. Anda dapat belajar untuk menggunakan Python dan merasakan manfaatnya segera dalam peningkatan produktivitas Anda.

Python berjalan pada Windows, Linux/Unix, Mac OS X, dan telah diporting ke Java (Jython) serta .NET (IronPython).

Python bebas untuk digunakan, bahkan untuk membuat produk komersial.

Ingin belajar Python?

  1. Download Python 2.7 atau Python 3
  2. Ikuti tutorialnya
